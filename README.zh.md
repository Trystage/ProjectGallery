# ProjectGallery

一个用于展示你最佳项目的网站。

## 语言

For English version, please see [README.md](README.md)

## 演示

访问我们的演示网站: [gallery.trystage.cn](https://gallery.trystage.cn)

## 功能

- 响应式设计，适配各种设备
- 美观的卡片布局
- 易于添加和管理项目
- 支持GitHub徽章显示星标和分支数

## 使用方法

1. 克隆或下载此仓库
2. 修改 `index.html` 文件添加你的项目信息
3. 根据需要自定义 `styles.css` 和 `script.js`
4. 部署到你喜欢的静态网站托管服务

## 项目结构

```
ProjectGallery/
├── index.html
├── styles.css
├── script.js
└── README.md
```
